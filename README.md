# 📋 Digital Notice Board System

A modern, fully functional **Digital Notice Board System** built with vanilla HTML, CSS, and JavaScript. Designed for offices, schools, hospitals, and organizations to replace traditional cork boards with an interactive, real-time digital solution.

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🎨 **Glassmorphism UI** | Eye-catching translucent design with animated gradients and smooth transitions |
| 📱 **Fully Responsive** | Seamless experience across desktops, tablets, and mobile devices |
| 🔔 **Smart Categories** | Color-coded badges for Urgent, Event, Announcement, Meeting, and General notices |
| 📌 **Pin System** | Pin critical notices to the top with visual pin indicator |
| 🚨 **Emergency Alerts** | Full-screen flashing emergency banner with pulse animation for critical announcements |
| 📊 **Live Dashboard** | Real-time statistics tracking total notices, urgent items, today's events, and view counts |
| 🔍 **Search & Filter** | Instant search by title/content/author; one-click category filtering |
| ❤️ **Social Features** | Like/unlike notices with per-user tracking |
| 📅 **Smart Scheduling** | Set event dates and auto-expiry for time-sensitive content |
| 🖼️ **Image Support** | Drag-and-drop image uploads with live preview |
| 📺 **Display Mode** | Full-screen presentation mode with auto-rotation, progress bar, and navigation controls |
| 🌤️ **Weather Widget** | Built-in weather display (ready for API integration) |
| 📰 **News Ticker** | Auto-scrolling marquee for urgent notices |
| 💾 **Persistent Storage** | All data saved to browser LocalStorage — no backend required |
| ⌨️ **Keyboard Shortcuts** | `Ctrl+N` new notice, `Escape` close modals, arrow keys in display mode |
| ⏰ **Auto-Cleanup** | Automatic removal of expired notices with background checks |

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Glassmorphism, CSS Grid, Flexbox, Keyframe Animations)
- **Logic:** Vanilla JavaScript (ES6+, Modular Architecture)
- **Storage:** Browser LocalStorage API
- **Icons:** Font Awesome 6.4.0
- **Dependencies:** Zero external JavaScript frameworks

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/digital-notice-board.git

# Navigate to directory
cd digital-notice-board

# Open in browser (or serve with any static server)
open index.html
```

No build step. No npm install. Works immediately in any modern browser.
